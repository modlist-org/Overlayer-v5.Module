# 🧩 Overlayer-Module

Runtime-loaded modules for game alteration.

> [!WARNING]
> ### Standalone Execution Not Supported
> **This module cannot function independently.** It requires the core [Overlayer-v5](https://github.com/modlist-org/Overlayer-v5) environment to run.

## 🔧 Installation
The installation process for this module is exactly identical to the main mod setup. Extract the package directly into your game's root directory, ensuring the folder structure aligns perfectly with the base mod environment.
